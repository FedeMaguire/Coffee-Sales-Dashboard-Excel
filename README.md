# Key Features:

## Data Gathering, Cleaning, and Transformation:
Showcase the skills in data acquisition and the adept use of Excel using Lookup and Index functions, pivot tables, for efficient cleaning and transformation of raw data related to coffee sales.
## Descriptive Statistics and Insights:
Employ Excel's powerful functions to generate comprehensive summary statistics, providing valuable insights into the sales performance of each coffee type.
## Pivot Tables and Pivot Charts:
Leverage Excel's robust pivot table and charting capabilities to dynamically explore and visualize coffee sales data. This enables intuitive insights and facilitates the discovery of trends for each coffee type.
## Interactive Dashboard Creation:
Designing an interactive dashboard that allows stakeholders to dynamically interact with the coffee sales data, providing real-time insights and custom views for all four coffee types.
